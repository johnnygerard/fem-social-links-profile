## Tech Stack

### Frontend

- **React Framework**: [Next.js 15](https://nextjs.org/)
- **Style**: [Tailwind CSS 4](https://tailwindcss.com/)

### Backend

- **CDN**: Static assets served from [Vercel Edge Network](https://vercel.com/docs/edge-network/overview).
